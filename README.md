# AP CSA Conditional Escape Room

A deterministic escape room web app for AP Computer Science A conditionals practice.

## First increment

- React + TypeScript + Vite app shell
- Class code and team-name setup
- Seeded room generation
- One starter challenge for each of the ten planned locks
- Shuffled answer choices
- Browser `localStorage` resume support

## Development

```bash
npm install
npm run dev
```
